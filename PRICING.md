# Dantian Agent Certification: Pricing and Cost Model

## Overview

Two-sided marketplace. Agent builders pay for certification. Users verify for free. Users pay a small fee for a human-readable Critical Report.

---

## Tier 1: Single Certification -- $49

One-time payment. One full audit. One 30-day certification window.

**What the agent owner gets:**
- Full audit: identity, capability, safety, compliance checks
- Structured certification report (JSON, stored on 0G Storage)
- On-chain certification with 30-day expiry
- Leaderboard listing while certified
- Searchable on the public verification frontend

**What happens after 30 days:**
- Certification expires on-chain. No renewal. No grace period.
- Agent disappears from the leaderboard.
- Frontend shows "Certification Expired" if queried.
- To re-certify: pay $49 again. Fresh audit, fresh report, fresh 30 days.

**No upsell during checkout.** One price, one scope, one month.

---

## Tier 2: Continuous Certification -- Prepaid Blocks

The agent owner pays upfront for a fixed block. Dantian runs automated re-audits every 24 hours. The certification stays current as long as the agent keeps passing.

| Block | Price | Per Month | vs. Tier 1 Equivalent | Savings |
|-------|-------|-----------|----------------------|---------|
| 3 months | $79 | $26.33 | $147 (3 x $49) | 46% |
| 6 months | $149 | $24.83 | $294 (6 x $49) | 49% |
| 12 months | $249 | $20.75 | $588 (12 x $49) | 58% |

**How it works:**
1. Agent owner pays upfront. Selects 3, 6, or 12-month block.
2. Dantian monitors the agent's source (GitHub repo, API endpoint, or declared identifier) on a 24-hour cycle.
3. Every 24 hours: pull latest state, run automated re-audit, compare against previous report.
4. If the agent passes, the on-chain certification expiry rolls forward by 24 hours.
5. If the agent fails, the certification is revoked immediately. The owner is notified via email. No refund for remaining prepaid time.
6. When the prepaid block ends, the cycle stops. The last certification expires normally (24h after last successful audit).

**No auto-renewal.** When the block ends, it ends. The owner can purchase a new block at any time.

**The selling point:** "Certified as of today, not as of last month."

**Revocation on failure (stated in terms):** If the agent fails a daily re-audit, certification is revoked immediately. Remaining prepaid time is forfeited. This is the cost of continuous trust; agents must maintain quality to keep their certification. This must be stated clearly in checkout flow and terms of service.

## Consumer Verification

### Free Check

Anyone can query the contract or use the frontend to see:
- Is this agent certified?
- What type of certification?
- When does it expire?
- Who certified it?
- What was the overall trust score?

This is free. Always. It is the public utility layer. No paywall on basic trust.

### Paid Critical Report -- $4

When a consumer is evaluating whether to use or download an agent, they pay $4 for a Critical Report.

**What a Critical Report contains:**
- Agent name, identifier, and source (repo URL, endpoint, etc.)
- What the agent claims to do (from its README, manifest, or declared capabilities)
- Plain-language summary of the last audit: what passed, what failed, what scored low
- Trust score (0-100) with a one-sentence explanation
- When the last audit happened and when certification expires
- A single recommendation:
  - "Safe to use for [declared purpose]"
  - "Use with caution: [specific concern]"
  - "Not recommended: [reason]"
- Download as PDF or static HTML

**What a Critical Report is NOT:**
- Not a new audit. It reads the existing certification data and the most recent full report.
- Not the full certification report. It is a simplified, human-readable summary.
- Not available if no certification exists. Returns: "No certification found. This agent has not been verified by Dantian."

**Generated by the Light Auditor agent** (see `light-auditor/README.md`).

## Cost Model

### Per-Review Cost Breakdown

All estimates use published Anthropic API pricing as of April 2026.

#### Full Audit (Tier 1)

The certifier agent runs a complete evaluation: analyze the agent source, test each claimed skill, run safety probes, generate the structured report.

| Cost Component | Tokens (Input) | Tokens (Output) | Unit Price | Cost |
|---------------|----------------|-----------------|------------|------|
| Agent analysis (repo clone, API probe, research) | 20,000 | 5,000 | Sonnet: $3/$15 per 1M | $0.135 |
| Skill evaluations (5 skills) | 10,000 | 3,000 | Sonnet: $3/$15 per 1M | $0.075 |
| Safety probes (jailbreak, PII, boundary) | 5,000 | 2,000 | Sonnet: $3/$15 per 1M | $0.045 |
| Report generation and verdict | 3,000 | 2,000 | Sonnet: $3/$15 per 1M | $0.039 |
| **LLM subtotal** | **38,000** | **12,000** | | **$0.294** |
| 0G Storage (report upload, ~8 KB) | | | Mainnet estimate | $0.001 |
| 0G Chain gas (certify tx) | | | Testnet: free; mainnet est. | $0.010 |
| Human review (15 min, certifier spot-check) | | | $50/hr effective rate | $12.500 |
| **Total per Tier 1 audit** | | | | **$12.805** |

**Why human review:** Tier 1 is the initial certification. The automated audit runs first; a human certifier reviews the results for 15 minutes before issuance. This prevents false certifications and builds trust in the certification brand. The human cost dominates Tier 1.

**Tier 1 margin:**
- Revenue: $49.00
- Cost: $12.81
- Gross margin: **73.9%** (above 70% target)

#### Daily Re-check (Tier 2)

The automated cycle pulls the latest agent state, runs a delta-based re-audit against the previous report, and extends or revokes the certification.

| Cost Component | Tokens (Input) | Tokens (Output) | Unit Price | Cost |
|---------------|----------------|-----------------|------------|------|
| State diff (repo pull / API health check) | 5,000 | 1,000 | Sonnet: $3/$15 per 1M | $0.030 |
| Delta skill evaluation (compare to previous) | 5,000 | 2,000 | Sonnet: $3/$15 per 1M | $0.045 |
| Safety spot-check | 2,000 | 1,000 | Sonnet: $3/$15 per 1M | $0.021 |
| Report update and status decision | 3,000 | 1,500 | Sonnet: $3/$15 per 1M | $0.032 |
| **LLM subtotal** | **15,000** | **5,500** | | **$0.128** |
| 0G Storage (report update) | | | | $0.001 |
| 0G Chain gas (extend expiry tx) | | | | $0.010 |
| **Total per daily re-check** | | | | **$0.139** |

**Why Sonnet for daily checks:** Haiku ($0.034/day) would be cheaper but produces thinner evaluations. At the Tier 2 price points, Sonnet is affordable and maintains audit quality. Thin daily audits at high margin would undermine the "always verified" value proposition.

**Monthly infrastructure cost per agent:**
- Always-on cron service (Railway/Render): $5/month base, amortized across agents
- At 20 agents: $0.25/agent/month
- At 50 agents: $0.10/agent/month
- Conservative allocation: **$1.00/agent/month**

**Tier 2 monthly cost:**
- Daily re-checks: $0.139 x 30 = $4.17
- Infrastructure: $1.00
- **Total: $5.17/month**

#### Critical Report (Light Auditor)

Reads existing certification data and most recent full report. One LLM call.

| Cost Component | Tokens (Input) | Tokens (Output) | Unit Price | Cost |
|---------------|----------------|-----------------|------------|------|
| Full report JSON + metadata + instructions | 3,400 | 500 | Haiku: $0.80/$4 per 1M | $0.005 |
| **Total per Critical Report** | | | | **$0.005** |

**Critical Report margin:**
- Revenue: $4.00
- Cost: $0.005
- Gross margin: **99.9%**

The high margin is justified: the value is in the certification framework and trust signal, not the raw compute. At $4, it is an impulse purchase for a developer evaluating an agent. The alternative (doing your own research) costs hours.
